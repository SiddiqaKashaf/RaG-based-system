## Tips for Inserting Images:

1. **Image Quality**: Use high-resolution screenshots (at least 1920x1080 or higher)
2. **File Format**: Use PNG or JPG format for screenshots, SVG or PNG for diagrams
3. **Caption Format**: 
   - "Figure X.Y: Description of the Image"
   - Place caption below the image, centered
4. **Sizing**: Images should be appropriately sized (not too large, not too small)
   - Full page width: For architecture diagrams and ERD
   - Half page width: For UI screenshots
   - Quarter page width: For small diagrams or icons
5. **Referencing**: Reference figures in text like: "As shown in Figure 4.4, the login page provides..."
6. **Page Placement**: Try to place images on the same page as their references or on the following page
